
# Checklist de Documentos Pessoais — Versão Premium Gamificada

Aplicativo web moderno, responsivo e gamificado para controle de documentos obrigatórios, com experiência visual de alto nível.

## Objetivo
Transformar o checklist em uma experiência intuitiva, motivadora e agradável, com sensação de progresso e recompensa visual.

## Novidades e Funcionalidades

- **Dashboard inicial** com resumo do progresso, total, pendentes, concluídos e próximos mais importantes
- **Modal de detalhes**: clique no ícone de informação para ver explicações detalhadas de cada documento
- **Barra de progresso animada** e percentual concluído
- **Mensagens motivacionais** dinâmicas conforme o avanço
- **Gamificação**: medalha/troféu visual ao concluir tudo, efeitos de conquista ao marcar itens
- **Sistema de prioridade** (alta, média, baixa) com cores e indicadores visuais
- **Separação visual clara**: pendentes, em andamento, concluídos
- **Observações por item** (salvas automaticamente)
- **Reset inteligente** com confirmação
- **Visual premium**: design profissional, elegante, excelente no celular, pronto para PWA/WebView
- **Responsividade total** (mobile-first)
- **Salva tudo automaticamente** no navegador (localStorage)

## Como usar
1. Abra o arquivo `index.html` no navegador.
2. Veja o dashboard inicial com resumo.
3. Clique no ícone de informação para ver detalhes de cada documento.
4. Marque os documentos conforme for providenciando.
5. Adicione observações em cada item.
6. O progresso é salvo automaticamente.
7. Ao concluir tudo, ganhe uma medalha visual!

## Tecnologias
- HTML, CSS, JavaScript puro (sem frameworks)
- Salva tudo no localStorage

## Experiência
Design moderno, elegante, lúdico e motivador. Excelente usabilidade no celular.
